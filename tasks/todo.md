### Реализация (Вариант: Vercel)

1. **API endpoint**: Уже готов `/api/cron/fetch-news` (есть SSE)
2. **Автоматизация**: Добавить `vercel.json` с cronjob:
   ```json
   {
     "crons": [
       {
         "path": "/api/cron/fetch-news",
         "schedule": "*/30 * * * *"
       }
     ]
   }
   ```
3. **Фронтенд**: Добавить кнопку в админ-панель, которая:
   - Делает запрос к `/api/cron/fetch-news` с заголовком `Authorization: Bearer <CRON_SECRET>`
   - Слушает SSE для получения логов в реальном времени
   - Показывает прогресс/статус

### Переменные окружения для Vercel

- `POCKETBASE_URL` - URL PocketBase
- `POST_LIMIT` - сколько постов с канала
- `TG_API_ID`, `TG_API_HASH` - Telegram API credentials
- `CRON_SECRET` - секрет для защиты API
